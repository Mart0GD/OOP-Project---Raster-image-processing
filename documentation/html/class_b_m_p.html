bool BMP::saveImage ( const Image & image,
const std::string & filename = NO_SAVE_NAME ) const
overridevirtual

Saves the image to a BMP file in binary format.

The BMP format is padded so that each row's size in bytes is a multiple of 4. A new BMPHeader is constructed during the save operation based on the image's properties.

Parameters
imageThe image to be saved.
filenameThe output filename. If not specified, a default name is used.
Returns
true if the image is successfully saved; false otherwise.
