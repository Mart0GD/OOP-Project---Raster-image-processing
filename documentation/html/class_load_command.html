Detailed Description

Note
a session can be started without any images, they can be added later on with AddCommand

Executes the command by loading and adding images to the session.

Reads the filenames from the input stream, loads the corresponding image files using the FileFactory, creates the Images, and adds them to the session one by one. Handles and reports all potential failures during this process. Clears all dynamically allocated memory on failure
