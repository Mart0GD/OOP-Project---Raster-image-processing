Parameters

posXThe x-coordinate where the image should be pasted.
posYThe y-coordinate where the image should be pasted.
Returns

Example

// image1 --> bmp
// image2 --> pgm
Image* pasted = image1.paste(image2, 0, 0); // Pasted will be a bmp
