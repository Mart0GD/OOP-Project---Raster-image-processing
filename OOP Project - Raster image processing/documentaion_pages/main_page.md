@section intro_sec Introduction
This project is a console-based raster image processing system for handling various image formats (PBM, PGM, PPM, BMP).

It supports sessions, command execution, image transformations, and file handling, implemented using design patterns like **Factory**, **Command**, and **Singleton**.
